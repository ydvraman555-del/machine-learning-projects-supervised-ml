# Clean Energy Dominance Prediction using Logistic Regression

This project focuses on predicting whether **clean energy is dominant** using a binary classification approach based on global energy and sustainability data.

## Project Overview
The objective of this project is to analyze renewable and fossil fuel energy indicators and build a classification model to predict the **clean_energy_dominant** target variable.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Workflow
- Data loading and preprocessing
- Handling missing values
- Encoding categorical variables
- Feature scaling
- Train-test split
- Model training and evaluation

## Model Used
- Logistic Regression (Binary Classification)

## Evaluation Metrics
- Accuracy Score
- Precision
- Recall
- Confusion Matrix
- Classification Report

## Results & Insights
The Logistic Regression model effectively classified whether clean energy dominance exists based on input features, providing meaningful insights into energy transition patterns.

## Conclusion
This project strengthened my understanding of **classification problems**, **Logistic Regression**, and evaluation techniques for binary outcomes in real-world datasets.
