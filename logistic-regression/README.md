# Clean Energy Dominance Prediction using Logistic Regression

This project focuses on predicting **Clean Energy Dominance** using a **Logistic Regression classification model** applied to real-world global energy data.

---

##  Project Overview
The objective of this project is to analyze renewable and fossil fuel energy indicators and build a classification model that predicts whether **clean energy is dominant** for a given region and time period.

---

##  Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

---

##  Workflow
- Data loading and understanding  
- Data cleaning and preprocessing  
- Handling missing values  
- Exploratory Data Analysis (EDA)  
- Feature encoding and scaling  
- Train-test split  
- Model training and evaluation  

---

##  Model Implemented
- Logistic Regression  

---

##  Evaluation Metrics
- Accuracy Score  
- Confusion Matrix  
- Precision, Recall, F1-Score  
- ROC Curve  
- ROC–AUC Score  

---

##  Results & Insights
- Logistic Regression achieved stable classification performance on the test dataset.  
- ROC–AUC analysis provided a better understanding of model performance across probability thresholds.  
- Threshold tuning demonstrated how prediction outcomes change based on different decision cutoffs rather than relying only on default thresholds.
