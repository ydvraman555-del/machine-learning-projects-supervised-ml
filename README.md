# 🤖 Machine Learning Projects

This repository contains my Machine Learning projects created while learning and implementing core ML algorithms using Python and Scikit-learn.

## 📂 What's Inside

In this repository, I have implemented the following Machine Learning algorithms:

- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Naive Bayes

## 📌 What I Did

For each project, I followed a complete Machine Learning workflow:

- Loaded and explored the dataset
- Cleaned and preprocessed the data
- Performed Exploratory Data Analysis (EDA)
- Applied feature engineering where required
- Split the data into training and testing sets
- Trained the Machine Learning model
- Evaluated the model using appropriate performance metrics
- Generated predictions and analyzed the results

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Scikit-learn
- Jupyter Notebook

## 🎯 Purpose

The goal of this repository is to strengthen my understanding of Machine Learning concepts through hands-on implementation and build a strong foundation in predictive modeling and data analysis.
